# Task Tracker Lite

A minimalist task management application built with the MERN stack.

**Live Demo:** [https://task-tracker-litebd.netlify.app/](https://task-tracker-litebd.netlify.app/)

## Features
- Create, edit, and complete tasks
- Soft delete functionality (keeps records in DB)
- Filter by status (All, New, Completed)
- Real-time search with debouncing
- Automatic date & time tracking for all actions

## Tech Stack
- **Frontend:** React, Tailwind CSS, Vite
- **Backend:** Node.js, Express, MongoDB

## Local Setup

### Backend
1. `cd backend`
2. `npm install`
3. Create a `.env` file with your `MONGODB_URI`
4. `npm start`

### Frontend
1. `cd frontend`
2. `npm install`
3. `npm run dev`
